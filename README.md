# Bug Summary

**Title**

Checkout Information accepts whitespace-only values.

**Severity**

Medium

**Priority**

Medium

**Status**

Open

---

## Expected Result

The application should reject whitespace-only inputs and display a validation message indicating that required fields cannot contain only spaces.

---

## Actual Result

The application accepts whitespace-only values and proceeds to the next checkout step without validation.

---

## Business Impact

Allowing whitespace-only inputs may lead to poor data quality, incomplete customer information, and inaccurate order records.
